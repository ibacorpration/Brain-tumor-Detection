Deep Learning Medical Imaging Project
An AI-powered medical imaging system that detects brain tumors from MRI scans using a deep learning model based on the EfficientNet architecture and built with TensorFlow / Keras.
The goal of this project is to assist in early detection of brain tumors and support medical decision-making through automated image classification.
The trained model analyzes MRI images and predicts whether a tumor exists and identifies its type.

---
#🧬 Tumor Classes

The model classifies MRI brain scans into four categories:

- 🧠 **Glioma Tumor**
- 🧠 **Meningioma Tumor**
- 🧠 **Pituitary Tumor**
- ✅ **No Tumor**

---

#📊 Model Performance

- Test Accuracy: 92.37%
